Namespaces

But, you don't want to have functions floating around in the global namespace, so you can create namespaces... 

var demo = demo || {};

demo.sayHello = function(name) {
    alert("Hello " + name + " from the demo namespace");
}

To call the <code>demo.sayHello("Jim") <code> function, click <a href="javascript:demo.sayHello('Jim');">here</a>.

Notice this construct : var demo || {};. Firstly we must use var to declare a truly global variable. Otherwise it will be implicitly added to the current global root object, which in a browser is window. The second thing to notice is the || 'or' expression.
